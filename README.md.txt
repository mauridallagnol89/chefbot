# ChefBot

Um assistente culinário inteligente integrado ao WhatsApp via Twilio e Google Gemini.

## Funcionalidades
- Sugere receitas com base nos ingredientes informados
- Dá dicas de preparo e substituições
- Personalidade de um chef experiente e simpático

## Tecnologias
- Flask
- Twilio API para WhatsApp
- Google Gemini AI
